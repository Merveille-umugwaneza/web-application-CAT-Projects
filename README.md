## Project 3: Hotel Website

A complete 6-page hotel website with database integration.

### Pages Included:
- **Home** – Main landing page
- **About Us** – Hotel information
- **Menu** – Food items displayed in an HTML table (Fish, Drinks, Fresh Juices)
- **Order** – Online food ordering form (stores data in database)
- **Gallery** – 5+ food/drink images (each image links to Order page)
- **Contact Us** – Message form (stores data in database)

### Additional Features:
- **Login System** – View customer orders after login
- **Logout Link** – End user session

### Database Stores:
- Customer order information (Name, Email, Phone, Menu selection, Address, Date)
- Contact messages (Name, Email, Phone, Location, Message)

**Tech used:** HTML, CSS, JavaScript, PHP, MySQL

## How to Run

1. Install XAMPP/WAMP on your computer
2. Copy project folder to `htdocs` (XAMPP) or `www` (WAMP)
3. Start Apache and MySQL
4. Import the database file into phpMyAdmin
5. Open browser and go to `http://localhost/your-project-folder`
